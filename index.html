实习两个月总结

发表于 2020-05-25

编程思路上

  1. 写页面顺序：先分析页面的核心功能有哪些，想想用户的使用顺序是怎么样的
  2. 页面的书写：
    1. 表格的布局，数据的渲染显示
    2. 功能实现步骤：
      1. 确定参数，先定义好参数
      2. 初始化参数
      3. 拿到后台数据
        • 注意检查数据是不是为空，校验
        • 将数据渲染到页面上
        • 做完需要简单测试，参数是否正确传到了
      4. 注意：
        • url里一般不放参数，get请求不安全
        • 时间参数推荐用‘-’分隔，‘/’可能会和转义符冲突，注意参数格式前后一致
        • 注意去掉参数前后空格（trim方法）
        • 减少ajax的使用

工作常用的工具

miniUI

  1. 页面一加载就使用遮罩mini.mask({})，完成ajax就取消遮罩效果mini.unmask()

    查询设计器

后台

  1. 怎么把数据在aspx里拼接好，前台可以一次性渲染出来？（4.7）

    浏览器调试

  2. 可以使用 console.log() 、alert()方法在调试窗口上打印 JavaScript 值
  3. 打断点，查看出问题的在哪一行。在每个断点上，都会停止执行 JavaScript 代码，以便于我们检查 JavaScript 变量的值。
  4. debugger 关键字用于停止执行 JavaScript，并调用调试函数。这个关键字与在调试工具中设置断点的效果是一样的。如果没有调试可用，debugger 语句将无法工作。
    1. 开启 debugger ，代码在第三行前停止执行。
  5. 注释部分代码，看哪些代码可以正确运行，就说明这一小部分没问题。找其他地方的问题
  6. 注意nerwork中查看ajax请求的地址是否正确，返回的数据是否正确

    E-charts

  7. 如果父级容器的height/width属性设置为百分比的形式，那么echarts就会warning，且不能正常的生成图表。所以div容器的高度宽度必须指定为px

IE8的兼容问题

  1. 不支持nth-child特性。解决办法：使用:first-child+子代标签

Hexo+Github搭建博客

发表于 2020-05-22

  1. 安装Node.js

  2. 安装Git

  3. 安装Hexo

    1. 在合适的地方新建一个文件夹，用来存放自己的博客文件，

    2. 在该目录下右键点击Git Bash Here

    3. 定位到该目录下，输入npm i hexo-cli -g安装Hexo

    4. 安装完后输入hexo -v验证是否安装成功

    5. 输入hexo init初始化文件夹，接着输入npm install安装必备的组件。

    6. 这样本地的网站配置也弄好啦，输入hexo g生成静态网页，然后输入hexo s打开本地服务器，然后浏览器打开http://localhost:4000/，就可以看到博客

  4. 连接Github与本地

    1. 在该目录下右键点击Git Bash Here
    2. 输入:
      • git config --global user.name "yehaha000"
      • git config --global user.email "email@qq.com"
    3. 生成密钥SSH key：ssh-keygen -t rsa -C "email@qq.com"
    4. 打开github，在头像下面点击settings，再点击SSH and GPG keys，新建一个SSH，名字随便。
    5. git bash中输入:cat ~/.ssh/id_rsa.pub
    6. 将输出的内容复制到框中，点击确定保存。
    7. 输入ssh -T git@github.com，出现你的用户名，说明成功了。
    8. 打开博客根目录下的_config.yml文件，修改最后一行的配置：

  5. 写文章

    1. 在博客根目录下右键打开git bash，安装一个扩展npm i hexo-deployer-git。
    2. 输入hexo new post "article test"，新建一篇文章
    3. 根目录下输入hexo g生成静态网页，然后输入hexo s可以本地预览效果，最后输入hexo d上传到github上。

  6. 绑定域名

    1. github学生包，name.com的免费域名一年
    2. ping yehaha000.github.io，将获得的IP地址添加到DNS记录里
    3. 打开github博客项目，点击settings，拉到下面Custom domain处，填上域名，保存
    4. 这时候你的项目根目录应该会出现一个名为CNAME的文件了。如果没有的话，打开你本地博客/source目录，我的是D:\study\program\blog\source，新建CNAME文件，注意没有后缀。然后在里面写上你的域名，保存。最后运行hexo g、hexo d上传到github。

  7. 安装Next主题

    1. 在博客根目录下右键打开git bash，git clone https://github.com/iissnan/hexo-theme-next themes/next
    2. 到站点配置文件里去设置一把，找到 theme 字段，并将其值更改为 next。(站点配置文件位于站点根目录下，主要包含 Hexo 本身的配置)
